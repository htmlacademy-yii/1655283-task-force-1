My Classes
---------------------------------------
ShowActions.php - Определяет возможные действия с заданием.
ShowTask.php - Содержит методы для вывода заданий по заданым параметрам
ShowUser.php - Содержит методы для вывода информации о пользователе/-лях.
ShowResponces.php - Содержит методы для вывода откликов на задание

TaskAdd.php - Методы для добавления задания
TaskUpdate.php - Методы для редактирования задания
TaskComments.php - Методы для работы с комментариями к заданию и вывода коментариев.
TaskResponce.php - Методы для добавления, обновления отклика на задание
TaskClose.php - Методы для завершения задания.
TaskCancel.php - Методы для отмены задания заказчиком.
TaskCancellation.php - Методы для отказа от выполнения исполнителем.
TaskStart.php - Методы для старта выполнения задания

UserReg.php - Методы для регистрации пользователя
UserDel.php - Методы для удаления профиля
UserLogin.php - Методы для входа на сайт
UserComments.php - Методы для работы и вывода комментариев к профилю пользлователя.
UserUpdate.php - Методы для работы с профилем пользователя

MSG.php - Методы для работы с внутренней перепиской на сайте
LOC.php - Методы для проверки и уточнения локаций (город, улица, координаты)
Events.php - Методы для работы с событиями
Imager.php - Методы для работы с изображениями пользователя
